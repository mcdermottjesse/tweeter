# Tweeter Project

Tweeter is a simple, single-page Twitter clone.

Tweeter implements HTML, CSS, JS, jQuery and AJAX front-end, aswell as Node, Express and MongoDB back-end.

#Getting Started

Clone Repo and Run

npm install
npm run local
visit the following URL

localhost:8080/
